League Rules

Group Stage

Every team plays every other team twice: one home game and one away game.

Gamedays

Fixtures are grouped into gamedays so each team plays once per round where possible.

Points

Win = 3 points, draw = 1 point, loss = 0 points. Ranking uses points, goal difference, then goals scored.

Playoffs

Top 4 qualify. Semifinals are two legs. If aggregate is tied, decide by penalties.

Final

The final is a single match and this format is the same for every season.
